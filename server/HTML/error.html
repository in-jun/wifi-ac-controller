오류 발생

에어컨 제어를 실패했습니다. 다음 사항을 확인해보세요:

  1. ESP8266 모듈이 제대로 동작하는지 확인하세요. 내부 LED가 켜지는지 확인하거나 모듈의 연결 상태를 점검합니다.
  2. ESP8266 모듈과 클라이언트가 같은 Wi-Fi 네트워크에 연결되어 있는지 확인하세요.
  3. 모듈을 재부팅해보세요. 모듈을 끄고 다시 켜는 것으로 문제가 해결될 수 있습니다.
  4. 모듈의 내부 LED가 정상적으로 동작하는지 확인하세요. LED가 켜진 상태면 모듈이 동작 중임을 나타냅니다.

문제가 지속된다면 여기를 클릭하여 이슈를 제보해주세요.
